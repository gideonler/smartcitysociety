# Hugo configurations

## Setup
Adding a new post
```bash
hugo new posts/"post name".md
```

## Usage
Running the server 
```bash
NPM Start
```

Building static pages
```bash
Hugo -D
```
Go to [http://localhost:1313/](http://localhost:1313)] to view static pages


Design changes to be made in the config.toml file
